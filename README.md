# Microsoft Power Platform (microsoft-power-platform)
Microsoft Power Platform provides low-code development tools with APIs for Dataverse data access, platform administration, and custom connector development.

## APIs

### Microsoft Dataverse Web API
The Microsoft Dataverse Web API provides OData v4 RESTful access to the Dataverse data platform that underpins Power Platform.

**Human URL:** [https://learn.microsoft.com/en-us/power-apps/developer/data-platform/webapi/overview](https://learn.microsoft.com/en-us/power-apps/developer/data-platform/webapi/overview)

### Power Platform Admin API
The Power Platform Admin API enables programmatic management of Power Platform environments, connectors, and governance policies.

### Power Platform Connectors
Power Platform Connectors provide pre-built integrations with hundreds of external services and enable custom connector creation.
