The Dataverse plugin for coding agents enables AI to safely access database tables, columns, and security settings with guardrails. Three scenarios showcase the plugin's capabilities: a developer building her first data model and app, a RevOps analyst running CRM queries in natural language, and a platform admin configuring security across regions.
